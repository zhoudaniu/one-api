# UI/UX Pro Max 技能

## 技能描述
专业的 UI/UX 设计和优化技能，专注于现代、美观、用户友好的界面设计。

## 核心能力

### 1. 设计系统创建
- 建立统一的设计语言和组件库
- 定义颜色系统、字体规范、间距系统
- 创建可复用的 UI 组件

### 2. 现代 UI 模式
- 渐变和玻璃拟态（Glassmorphism）
- 微交互动画和过渡效果
- 卡片式布局和网格系统
- 响应式设计适配

### 3. 用户体验优化
- 直观的导航结构
- 清晰的视觉层次
- 无障碍访问（a11y）支持
- 加载状态和反馈机制

### 4. 技术实现
- React/Vue 组件化开发
- CSS-in-JS 或 Tailwind CSS
- 动画库集成（Framer Motion、GSAP）
- 性能优化

## 使用方法

### 分析现有 UI
```
分析当前页面的 UI 设计，识别改进点
```

### 生成设计规范
```
为项目创建完整的设计系统规范
```

### 重构页面
```
使用现代 UI 模式重构指定页面
```

### 优化用户体验
```
优化页面的交互流程和用户路径
```

## 设计原则

1. **简洁性** - 去除不必要的元素，保持界面清爽
2. **一致性** - 统一的视觉语言和交互模式
3. **反馈性** - 及时的用户操作反馈
4. **可访问性** - 支持所有用户群体
5. **性能** - 优化加载速度和渲染性能

## 输出格式

- 设计规范文档
- 组件代码实现
- 样式文件
- 使用示例
- 优化建议
